# brightspaceR: Access D2L Brightspace Data Sets via the BDS API

Connect to the D2L Brightspace Data Sets (BDS) API via OAuth2, download
all available datasets as tidy data frames with proper types, and join
them using convenience functions that know the foreign key
relationships.
